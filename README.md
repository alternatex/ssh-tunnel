Tunnelx
=============

SSH-Tunnel management utility

Prerequisites
-------------
Unix-OS

Installation
-------------

You can install this through [shinst](https://github.com/alternatex/shinst):

`shinst install alternatex/tunnelx -n tunnelx`

Usage
-------------

```bash
tunnelx
```
